# PPT Compactor — site

Public privacy policy and support page for [PPT Compactor](https://apps.apple.com/) — a native macOS app that strips unused media and stills out videos / animated GIFs in a `.pptx`.

App source is in a separate private repository. This repo exists only to host the public-facing Privacy Policy and Support pages required for App Store Connect submission.
