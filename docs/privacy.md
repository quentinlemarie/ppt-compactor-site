# Privacy Policy

Effective date: May 20, 2026

PPT Compactor is a local macOS utility. It does not collect, transmit, sell, or share personal data.

## What the app processes

When you choose a `.pptx` file, PPT Compactor reads that file on your Mac so it can identify embedded media and write a cleaned copy. The app may create temporary files while extracting still frames from videos or while copying selected media. Temporary files are removed automatically when the related operation finishes or when the document is reset.

## Network use

The app does not upload your presentations or media. It does not use analytics, crash reporting, advertising SDKs, tracking SDKs, or third-party network services.

## Files and permissions

PPT Compactor uses macOS user-selected file access. It can read files you explicitly open and write to locations you explicitly choose through the standard macOS file picker.

## Privacy rights

Because PPT Compactor does not collect personal data, it has no account profile, analytics record, advertising identifier, or server-side user record to access, export, sell, share, correct, or delete under GDPR, CCPA, or similar privacy laws. If you contact support, your message is handled only so the request can be answered.

## Contact

For privacy questions, use the support contact listed on the [Support](support.md) page.
