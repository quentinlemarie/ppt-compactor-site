# Support

For help with PPT Compactor, email:

**quentin.lemarie@gmail.com**

Please include:

- macOS version
- PPT Compactor version
- what you were trying to do
- the error message, if one appeared
- whether the presentation was created in Microsoft PowerPoint, Keynote, Google Slides, or another tool

Do not send confidential presentations unless explicitly requested. Most issues can be diagnosed from the app version, the file size, and the visible error message.

## Common checks

- Make sure the file is a real Microsoft PowerPoint `.pptx`, not a renamed `.docx`, `.key`, or `.pdf`.
- If a video cannot be converted to a still frame, keep the original video and save again.
- If a deck was exported from Keynote or another tool, stale internal references may remain. PPT Compactor preserves pre-existing stale references rather than deleting media that could still matter.
