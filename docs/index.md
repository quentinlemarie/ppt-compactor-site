# PPT Compactor

PPT Compactor is a native macOS utility for shrinking oversized Microsoft PowerPoint `.pptx` files before sharing or archiving them.

It opens a presentation locally, lists embedded media, and lets you:

- compact large used slide images
- remove unused media safely
- convert embedded videos and animated GIFs to still frames
- inspect where each media file is used, including whether it sits inside or outside the visible slide boundary
- export or copy selected media files

## Local-only by design

PPT Compactor works on files you choose on your Mac. It does not upload presentations, does not use analytics, and does not require an account.
